A New Balance

  • A new theoretical way to look at E-I balance
  • Unlike the E-I balance in balanced networks where the balance is a phenomena of the neural dynamics of the network, here the balance arises from the dynamics of synaptic plasticity.
  • It's not only new but also important because I hope to convince you that The new balance has an important computational role: Only balance weights can be robust in a noisy environment

E-I Balance in Associative Memory Networks

  • A good way to talk about this new balance is to reexamine the very well studied model for associative memory:
    • Recurrent network of binary units with some constant threshold
    • Asynchronous update
    • Memories are fixed points of the network
  • Implementing a set of given fixed points in a network is equivalent to solving a feed forward perceptron problem
  • We would like to see what happens in this system when we separate E and I
  • The strategy would be to analyze the single neuron properties and through that understand the implication for networks

Signal to Noise

  • Now let us do a simple SNR estimation for the two types of weights. (explain SNR)
  • After Signal: It seems that balanced weights are better.

  • Input Noise: Maybe now it is clear why the distinction wasn't made before:

    People were concerned with input noise for which there is no difference.

    In fact we will show that in many cases the most input noise robust weights are unbalanced

  • Conclusion: Only balanced solutions can be robust to both input and output noise

Only balanced synaptic weights can produce networks that are robust to both input and output noise.

This is the main point! If you want the network to be robust to both types of noise the weights must be balanced.

Especially when N is large since the SNR of unbalanced weights goes down with N.
